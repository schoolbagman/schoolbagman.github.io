检讨书

发表于 2017-02-23   |   分类于 日志   |     |   阅读次数

致我亲爱的杨晓君同学：

公园2017年2月23日下午20点40分左右，由于我一时得意忘形，在双方的友好磋商中说了不该说的话，对此，我深表后悔。为向我最爱的女朋友——杨晓君同学表明我的悔过决心，特书此检讨书。

首先，我先对我犯错的动机、时机进行分析。在通话过程中，由于想到明天差不多这个时候——写文章的这个时候，我们差不多就能见面了，内心十分激动。在激动之余，说话的时候口不择言，本来是向女朋友申请怜爱的陈述——“你看我去见一次多麻烦。”结果，在女朋友杨晓君同学开玩笑说：“那你就不来了。”可能出于我的低级趣味，认为有时候拌嘴会很好玩，同时内心期待她能够说不准我不来，她像我之类的，在之后的对话中，我都说：“好的。”当时我还以为自己的这招很受用。

结果没有想到其实刚开始在一起就已经异地恋对杨晓君同学来说已经很难，而且之前在情人节的时候，由于我的疏忽大意，已经导致了没有赶上火车前往广州与朝思暮想的杨晓君同学共度佳节——自己折腾是小，让杨晓君同学落得一场空欢喜，当时发生的时候，杨晓君同学不但没有生气，反而还安慰我说没关系——这让我很受感动，同时，可能也为今天的肆意妄言埋下了伏笔，他的体谅让我以为可以凭着她的温柔而肆意妄为，无所顾忌，随心随性。但是正如我一直相信的一样，爱情当中的付出与体谅应该是相互的，一味的付出与包容只会让另外一颗心受伤。

我觉得自己最过分的是当杨晓君同学说：“那我找其他男人和我住。”原本这话里已经包含了深深的不满，当时我的木鱼脑袋却没有意识到这层意思，其实我觉得我只要表达我的在乎以及之前说的全是屁话其实就好。但是我鬼使神差的说：“好的。”——如果换了我是杨晓君同学的话我也会狠狠的生气的。这句话背后原本的期盼是希望我的在乎，而我的回答恰好与这种期盼背道而驰，而且即使是玩笑，也应该是有分寸的。不能因为杨晓君的同学的包容而一步步的把底线往下压，她的不满的爆发是随时的，而我却不知道压死骆驼的最后一根稻草是自己什么时候放下的。

不过感谢上帝，她并没有彻底的不理我，我竟然还能打通她的电话。无论这童话过程中她的态度如何，我想这都是我应得的惩罚。从在一起的第一天开始，我就觉得她是上帝对我唯一的眷顾，我本配不上她，可还如何肆意践踏她的美丽善良——我真该下地狱。

但是她实在太好了，我的眼光没有错。

而且，我爱他。

为此，我愿意因为我的错误承担一切冷漠亦或者是惩罚——她太过温柔，竟不愿惩罚我。

从前没有，此后不会，只此一次。

晓君，原谅我，好吗？这真的太煎熬了。

曾诚，悔过于2017年2月23日。
